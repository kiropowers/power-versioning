# 版本控制助手

[English](README.md) | [中文](README.zh-CN.md)

Kiro Power，用于自动化版本控制和 CHANGELOG 生成，支持 Conventional Commits 规范。

## 功能

- 为你的项目配置 standard-version
- 自动生成 CHANGELOG.md
- Conventional Commits 提交规范指导
- 语义化版本管理

## 安装

在 Kiro IDE 中：

1. 打开 Powers 面板
2. 点击 "Add Power" → "From GitHub"
3. 输入：`https://github.com/kiropowers/power-versioning`

## 激活关键词

当你提到以下内容时，此 Power 会被激活：

- `版本` / `version`
- `changelog`
- `发布` / `release`
- `提交规范` / `conventional`

## 使用

直接问 Kiro：

- "帮我配置项目的版本控制"
- "这个提交消息应该怎么写？"
- "我应该发布什么版本？"

Kiro 会通过 steering 文件指导你完成整个流程。
