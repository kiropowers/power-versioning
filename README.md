# Versioning Power

[English](README.md) | [中文](README.zh-CN.md)

Kiro Power for automated versioning and CHANGELOG generation with Conventional Commits.

## Features

- Configure standard-version for your project
- Auto-generate CHANGELOG.md
- Conventional Commits guidance
- Semantic versioning management

## Installation

In Kiro IDE:

1. Open Powers panel
2. Click "Add Power" → "From GitHub"
3. Enter: `https://github.com/kiropowers/power-versioning`

## Activation Keywords

This Power activates when you mention:

- `version` / `版本`
- `changelog`
- `release` / `发布`
- `conventional` / `提交规范`

## Usage

Just ask Kiro:

- "Help me configure version control for my project"
- "How should I write this commit message?"
- "What version should I release?"

Kiro will guide you through the process using the steering files.
